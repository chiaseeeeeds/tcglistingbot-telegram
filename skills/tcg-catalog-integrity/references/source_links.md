# TCG Source Links

Use these as approved references when working on this repo's OCR, catalog, and pricing paths.

## Tooling

- CLI-Anything: https://github.com/HKUDS/CLI-Anything
- Scrapling: https://github.com/D4Vinci/Scrapling

## Catalog / Set Data

- Pokemon Card CSV: https://github.com/tradingcarddex/Pokemon-Card-CSV
- Bulbapedia Pokémon TCG expansions: https://bulbapedia.bulbagarden.net/wiki/List_of_Pok%C3%A9mon_Trading_Card_Game_expansions

## Pricing / Market Data

- PriceCharting: https://www.pricecharting.com/
- PriceCharting API docs / guide entry point: https://www.pricecharting.com/price-guide
- Pokémon TCG API: https://pokemontcg.io/

## How To Use These Links

- Prefer official/API sources before scraping.
- Use scraping only as a best-effort fallback when an API is unavailable or incomplete.
- Do not convert any one-off lookup result into a runtime hardcoded card mapping.
- If a source is blocked by Cloudflare or unavailable, surface that limitation explicitly.
